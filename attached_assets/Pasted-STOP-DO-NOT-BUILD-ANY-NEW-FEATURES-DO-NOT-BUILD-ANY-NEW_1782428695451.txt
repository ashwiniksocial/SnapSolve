3. WEAK STUDENT SIMULATOR

This is the most important module.

Pretend to be a student scoring 20 out of 100.

Read the lesson.

After every paragraph ask

"What am I confused about?"

Generate ALL possible confusion.

Examples

"What is p?"

"What is irrational?"

"What is contradiction?"

"What is coprime?"

"Why square both sides?"

"Why is p even?"

"Where did that equation come from?"

"What happened here?"

The simulator should continuously search for confusion.

If ANY confusion exists,

mark lesson as FAILED.
