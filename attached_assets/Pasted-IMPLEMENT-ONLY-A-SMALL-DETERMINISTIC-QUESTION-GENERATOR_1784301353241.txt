Objective:
Prove that deterministic offline templates can generate valid stored QuestionV2 records that behave exactly like existing handwritten questions.

   - clear numerical question patterns,

   - existing QuestionV2 questions,

2. Create one development-only generator script:
   scripts/src/generateMathQuestionProof.ts

3. Create exactly 3 deterministic templates, for example:
   - direct calculation,
   - application/word problem,
   - misconception-based MCQ.

4. Use a fixed seed or fixed parameter sets.
   Re-running the script must produce byte-for-byte identical question content and identical IDs.

5. Generate exactly 10 new QuestionV2 questions:
   - 4 Easy
   - 4 Medium
   - 2 Hard

   - permanent canonical IDs
   - no collision with existing IDs
   - valid answers
   - plausible unique MCQ options where applicable
   - complete steps[]
   - hint
   - keyConcepts
   - conceptsCovered
   - commonErrors
   - tags
   - all mandatory QuestionV2 fields

6. Do not overwrite or regenerate existing chapter questions.

7. Prefer writing the generated records into a separate proof file:
   question-bank/questions/mathematics/class<N>/<existing-chapter>-generated-proof.ts

8. Add only the minimum import/spread required to include these 10 questions in the existing Mathematics question pipeline.

9. Add one script entry to scripts/package.json only if required:
   "generate:math-proof": "tsx ./src/generateMathQuestionProof.ts"
